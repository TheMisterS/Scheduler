# Introduction

Scheduler is a small personal project to practise PostgreSQL and JAVA. <br>

It consists of a PostgreSQL database and an interface built with Java and the JDBC library. <br>


## Requirements

PostgreSQL 11 or later <br>
Java (JDK) <br>
